# वेद ज्ञान — Veda Gyan

Interactive Hindu Vedas explorer in Hindi + English.

## Deploy to Vercel (Static — no Python needed)

1. Push this repo to GitHub
2. Vercel → New Project → Import repo
3. Framework: **Other**
4. Output directory: `public`
5. Deploy ✅

## File structure

```
/
├── public/
│   └── index.html   ← The entire app (187 KB)
└── vercel.json      ← Static site config
```

No Python. No Flask. No build step. Just HTML.
